# outline-leyu-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖不同领域的信息参考与整理。  
每个页面均为独立构建，不针对特定域名或网站，仅作为通用性资料归档使用。

仓库命名中的 `leyu` 为项目代号，无特殊指向性。

## 目录结构

```
outline-leyu-guide/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 页面引用的静态资源（CSS、图片、字体等）
└── README.md        # 本文件
```

所有页面文件均放置在 `pages/` 目录下，按内容类型或编号进行组织。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开使用。
- 页面内容以信息整理、参考指引为主，不包含交互式服务或动态数据。
- 部分页面可能引用 `assets/` 目录下的资源文件，请确保文件路径完整。
- 页面标题和内容描述请参考各文件内部的注释说明。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面或优化现有内容。
- 欢迎通过 Issues 提交建议或指出内容问题，但请注意不要涉及具体域名或商业推广。
- 不接受针对特定网站或服务的定制化请求。
- 所有页面内容仅供个人学习与参考，请勿用于商业用途或违反相关法律法规。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的 `.html` 文件即可查看。
3. 若页面依赖资源文件，请确保 `assets/` 目录与 `pages/` 同级。
